## 订阅地址转换

[id9](https://id9.cc/)

把自己协议的订阅地址，输入进去转换后，替换surge4.conf中“订阅地址”4个字（出现了几遍，最好都替换一下）。

转换的时候，可能需要勾选“输出为 Node List”
